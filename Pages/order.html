To place an order:

  1. For CLEI labels, have your CLEI codes ready. If you need more information on determining CLEI codes, click here
  2. Download the orderform
  3. Fill out the required shipping, billing, and payment fields
  4. Fill out the following for each unique item:
    • Product identifier
    • Manufacturerâ€™s part number (e.g. 123 rev01)
    • Manufacturer 4 character code (as per ANSI T1.220)
    • CLEI code (e.g. SNCPN141AA)
    • 6 digit ECI (Equipment Catalog Item, e.g. 123456)
    • Label type as defined in GR-383-CORE
    • Quantity of labels required
    • For 2D labels a list of serial numbers along with the orderform is required
  5. Submit your orderform to sales@stogas.com
